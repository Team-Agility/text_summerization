system and reference summaries should be devoid of any punctuation marks
from the 'rouge2.0-distribution' directory, run 'java -jar rouge2.0.jar'
results will be availalbe in a text file named 'results.csv'
system summaries names need be in format 'meetingID_systemname.txt'
